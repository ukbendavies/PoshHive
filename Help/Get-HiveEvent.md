# Get-HiveEvent

## SYNOPSIS
Retrieves the latest set of events that have occurred on the Hive Api surface.

## SYNTAX

```
Get-HiveEvent
```

## DESCRIPTION
Uses the Hive Events Api to get the latest set of events that have occurred in your Hive system.
