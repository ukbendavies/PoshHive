# Get-HiveEvent

## SYNOPSIS
Retrieves the latest set of events that have occured on the Hive Api surface.

## SYNTAX

```
Get-HiveEvent
```

## DESCRIPTION
Uses the Hive Events Api to get the latest set of events that have occured in your Hive system.
